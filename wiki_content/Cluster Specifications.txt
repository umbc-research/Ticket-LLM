CPU and GPU Specifications
There are 145 nodes in the chip cluster divided into partitions. Please refer to this page for more information on the partitions: 
https://umbc.atlassian.net/wiki/spaces/faq/pages/1249509377
 
CPU Nodes
49 nodes purchased in 2018 that each have 2x 18-core Intel(R) Xeon(R) Gold 6140 CPU @ 2.30GHz (Skylake) with 376 GB of CPU memory and 72GB of local storage;
18 nodes purchased in 2021 that each have 2x 24-core Intel(R) Xeon(R) Gold 6240R CPU @ 2.40GHz (Cascade Lake) with 187 GB of CPU memory and 396GB of local storage; 
13 nodes purchased in 2024 that each have 2x 32-core Intel(R) Xeon(R) Gold 6548Y+ (Emerald Rapids) with 1024 GB of CPU memory and 1.8TB of local storage;
38 nodes purchased in 2024 that each have 2x 32-core Intel(R) Xeon(R) Gold 6548Y+ (Emerald Rapids) with 512 GB of CPU memory and 1.8TB of local storage.
CPU Node Schematics: 
https://umbc.atlassian.net/wiki/spaces/faq/pages/1333821495/Node+Schematics#CPU-Node-Schematics

4 nodes purchased in 2020, with 2x 24-core Intel(R) Xeon(R) Gold 6240R CPU @ 2.40GHz (Cascade Lake) with 384 GB of CPU memory, 8x RTX 2080Ti GPU cards each with 11 GB of GPU memory, and 1.9TB;
7 nodes purchased in 2020, with 2x 24-core Intel(R) Xeon(R) Gold 6240R CPU @ 2.40GHz (Cascade Lake) with 384 GB of CPU memory, 8x RTX 6000 GPU cards each with 24 GB of GPU memory, and 1.9TB;
2 nodes purchased in 2020, with 2x 24-core Intel(R) Xeon(R) Gold 6240R CPU @ 2.40GHz (Cascade Lake) with 768 GB of CPU memory, 8x RTX 8000 GPU cards each with 48 GB of GPU memory, and 1.9TB;
2 nodes purchased in 2024, with 2x 16-core Intel(R) Xeon(R) Gold 6548Y+ (Emerald Rapids) with 256 GB of CPU memory, 2x H100 GPU cards each with 100 GB of GPU memory and NVLink between them, and 7TB of local storage; 
10 nodes purchased in 2024, with 2x 16-core Intel(R) Xeon(R) Gold 6548Y+ (Emerald Rapids) with 256 GB of CPU memory, 4x L40S GPU cards each with 48 GB of GPU memory, and 7TB of local storage.

Networking Connections
The CPU cluster nodes are all connected to each other and 
enterprise storage
 by a high speed backend Infiniband network supporting 100 Gbps. The GPU cluster nodes are all connected to each other and 
enterprise storage
 by a high speed backend Ethernet network supporting 10 Gbps.
