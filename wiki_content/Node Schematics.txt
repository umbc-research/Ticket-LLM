CPU Node Schematics
Legend
Text (e.g., "2021 Nodes"):
 Node Generation/Year (2018/2021/2024)
Dashed Line Box:
 Node (Complete Computing Unit)
Text (e.g., "Xeon Gold 6240R"):
 CPU Model
Blue Rectangles:
 DDR4 or DDR5 Memory Modules (RAM)
Grey Box:
 CPU Socket
Lines between Grey Boxes:
 Intel UPI Connections (Inter-Socket Communication)
Red Rectangles:
 CPU Cores (number per socket specified by text, e.g., "Cores (24/Socket)")

GPU Node Schematics
Legend
Green Rectangles:
 GPU Card (e.g., H100, RTX 2080Ti)
Text on/next to GPU Card:
 GPU Memory (e.g., 100 GB HBM3)
Lines from CPU to GPU:
 PCIe Connection
Line directly between GPUs:
 NVLink Connection
